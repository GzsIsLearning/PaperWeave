# Inference Dynamics Analysis

## Overview

LLM 推理的内部动力学分析：从几何和信息论角度研究推理过程中内部表征的演化规律。核心发现是推理时的内部动态会自组织成低维流形，有效的推理需要满足特定几何条件（充分表征能力、自发流形压缩、保持非退化信息体积）。
